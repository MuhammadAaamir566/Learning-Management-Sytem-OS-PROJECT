# Learning Management System (Bash)

## Description
This is a CLI-based Learning Management System developed in Bash Script.
It supports three roles:
- Admin
- Teacher
- Student

## Features
- Role-based login system
- Course management
- Student enrollment
- Marks & grades management
- File-based database (TXT files)
